Why emojimage?

Self-Hostable

Complete control. Run the bot on your own server or machine using the simple bot.py entry point.

Unified Downloader

Easily grab both static and animated emojis, as well as official Discord stickers, with a single command.

FakeNitro Support

Even if you are using FakeNitro to send assets, emojimage can still detect and resolve the correct links for downloading.

How It Works

All operations occur locally on the machine where you host the bot. emojimage processes the following information strictly in real-time to execute commands:

  • Discord Message Content (to detect emoji/sticker links).
  • User IDs (to verify command permissions, if configured).

None of this information is logged, stored permanently, or transmitted to any external servers by the default emojimage installation.
